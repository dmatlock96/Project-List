# Airbnb Amentity Detection Project

## Purpose

Create a predictive model that, when given an image, will identify the amentities in that room.

## Motivation

Airbnb published an Medium article outlining how they built an ametity detection model; however, they did not publish the code pertaining to this. My goal is to recontruct their model from scratch using the article as a guide.

Airbnb Amentity Detection Article: https://medium.com/airbnb-engineering/amenity-detection-and-beyond-new-frontiers-of-computer-vision-at-airbnb-144a4441b72e
